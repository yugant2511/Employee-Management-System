Project Overview

The Employee Management System is a Java console-based application that allows organizations to manage employee records efficiently. It supports CRUD operations, fast searching, salary reporting, and persistent storage using file handling and serialization.

Features

Add, view, update, and delete employees

Search employees by ID, name, and department

Salary calculation and reporting

Data persistence using serialization

Exception handling for robustness

Technologies Used

Java (JDK 8+)

Java Collections Framework

File Handling & Serialization

Setup Instructions

Install JDK 8 or above

Clone the repository

Open the project in IntelliJ IDEA / Eclipse / VS Code

Compile all .java files inside src

Run EmployeeManagementSystem.java
